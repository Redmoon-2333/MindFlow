## 一、国际竞品分析

### 1. RescueTime — 自动追踪鼻祖

| 维度 | 详情 |
|------|------|
| 核心功能 | 自动记录应用/网站使用时间；按生产力评分分类（Very Productive / Productive / Neutral / Distracting / Very Distracting）；专注会话+应用拦截；Timesheets 模块自动生成项目时间报表 |
| 采集深度 | 桌面端全自动（应用名+URL+窗口标题）；iOS 可选 Screen Time 权限；无截图/无按键记录/无鼠标追踪 |
| 报告形态 | 日/周摘要仪表盘；按类别和时间段的可视化报表；Timesheets 可导出客户端 CSV/PDF |
| 干预手段 | Focus Sessions（时间盒+可选拦截）；智能休息提醒；「Focus Time」目标设定 |
| AI 能力 | 弱。基础自动分类（基于数据库匹配），无个性化 AI 建议。2025 年转向「实时专注引导和倦怠预防提醒」但仍为规则驱动 |
| 定价 | Solo Focus $12/mo（年付）；Solo Plus（含 Timesheets）$16/mo；Team Focus $10/seat/mo |
| 免费层 | 有限功能（14天追踪）+ 升级引导 |
| 用户抱怨 | 分类不够准确需手动修正；免费版限制过多；数据精度不足以支撑计费；界面偏旧 |
| 商业模式 | B2C + 小团队 B2B |
| 来源 | rescuetime.com; reddit.com/r/ProductivityApps "RescueTime rant" id=1iw7xfc |

**综合评价：** 自动追踪的标杆，但 AI 能力几乎为零，无根因分析能力。用户最大的痛点是「被追踪但不被理解」——知道时间去哪了，但不知道为什么。

### 2. Rize.io — AI 自动化时间追踪（最接近「AI教练」概念的产品）

| 维度 | 详情 |
|------|------|
| 核心功能 | 自动追踪桌面活动（应用名+URL）；AI 自动分类；专注检测；内置专注音乐；日程集成；AI 时间条目生成 |
| AI 教练 | 2024年发布「AI Coach」：Insights 标签页自动分析用户数据，提取个性化洞察（如 "You are most productive between 9-11am"）；智能休息提醒；AI 会话计划器 |
| 采集深度 | 同 RescueTime（元数据级别），MCP 服务端支持 |
| 定价 | Basic $12.99/mo（$9.99 年付）；Pro $28.99/mo（含 AI 洞察、PDF/CSV 导出、API/Webhook）；Business $49.99/mo；Team 从 $20/seat/mo 起 |
| 免费层 | 7天免费试用（无永久免费层） |
| 用户评价 | 正面：「从未需要问自己时间去哪了」。负面：无免费层；AI 洞察仍较基础（"insights are more descriptive than prescriptive"） |
| 来源 | rize.io/pricing; rize.io/changelog/releasing-a-i-coach-productivity-insights; reddit id=1jnst4v |

**综合评价：** 市场上「AI 教练」概念最完善的产品，但 AI 停留在「描述性分析」层面（what + when），没有触及「why you procrastinate」的根因分析，不做任何 CBT/心理层面干预。

### 3. Opal / Freedom / Cold Turkey — 强制拦截三强

| 维度 | Opal | Freedom | Cold Turkey |
|------|------|---------|-------------|
| 核心功能 | iOS 深度拦截 + AI 分析 | 全平台同步拦截，锁机模式 | 最强拦截力度，不可绕过 |
| AI 能力 | AI 分析使用模式 | 无 | 无 |
| 拦截策略 | Deep Focus 极难绕过 | Locked Mode 防提前结束 | 锁机后几乎不可终止 |
| 定价 | ~$59.99/年（仅 iOS） | $39.99/年（全平台） | 基础免费，Pro $39 买断 |
| 用户抱怨 | 仅 iOS；定价高；错过重要通知 | 同步问题；误伤工作网站 | UI 老旧；误拦截难解除 |
| 来源 | opal.so; getfaithlock.com "Freedom vs Opal 2026"; getcoldturkey.com |

**关键洞察：** 拦截类产品的核心问题是用户会「学会绕过」——新鲜感消退后卸载率极高。越强的拦截 = 越高的反感（立项书关键问题3的直接证据）。

### 4. Forest（专注森林）— 游戏化标杆

| 维度 | 详情 |
|------|------|
| 核心功能 | 番茄钟+虚拟种树（离开则树枯萎）；解锁树种；好友共种；排行榜 |
| 社会影响 | 与 Trees for the Future 合作，超 400 万付费用户 |
| 游戏化机制 | 损失厌恶；收集要素（41+ 树种）；专注金币兑换真实种树 |
| 定价 | iOS $3.99 买断；Android 免费/Pro $1.99 |
| 用户抱怨 | 更新缓慢；功能单一；离开 App 限制太绝对；中文版广告多 |
| 来源 | forestapp.cc; reddit.com/r/forestapp id=1hl7qeo; zhuanlan.zhihu.com/p/539297281 |

**关键洞察：** 高下载量但低 ARPU。游戏化是「开启」动机而非「持续」动机——「用了一周就腻了」。**没有产品在回答「为什么你停不下来」**。

### 5. Session / Sukha（原 Centered）— 流程状态导向

| 维度 | Session | Sukha |
|------|---------|-------|
| 核心功能 | iOS/Mac 极简专注计时+情绪反馈 | AI 教练+Flow 音乐+待办+社区共作+拦截一体化 |
| 定价 | 免费/Pro $4.99/mo | $20/mo 或 $400 终身 |
| 用户抱怨 | 功能有限 | 价格高；定位模糊；改名后老用户流失 |
| 来源 | productivity.directory/session; thesukha.co; reddit id=1g2npma |

**关键洞察：** Sukha 是功能形态上最接近的对标，但没有因果分析和 LLM 自适应干预。其从 $40/mo 降到 $20/mo 暗示一体化工具 B2C 付费意愿有天花板。

## 二、国内竞品分析

### 1. 番茄 ToDo
- 番茄钟+待办+锁机+统计+自习室；专业版 9.9 元/季度
- 抱怨：「自己瞎统计了一大堆，最后连每天最基本的任务都完不了」；锁机太绝对；强调时长而非效率
- **核心问题：计划对象是时间而非任务完成，用户陷入「凑番茄钟数」陷阱。无 AI 能力**
- 来源: zhihu id=370427247; lillianwho.com

### 2. 专注清单（Focus ToDo）
- 计划对象是「待办」而非番茄数；87 元永久/25 元每3月
- 抱怨：操作复杂；UI 不美观
- 来源: focustodo.cn

### 3. Forest 中文版
- Seekrtech 发行，付费下载后仍有广告的抱怨较多；活跃用户峰值 1800 万（百度百科）

### 4. 飞书/钉钉/企业微信
- 均无独立专注产品线，「勿扰模式」只是 IM 小模块
- **B2B 场景下「专注」尚未被视为独立付费需求**；巨头 AI 竞争集中在 AI+多维表格/助理
- 来源: feishu.cn; 21jingji.com/20251205; woshipm.com

## 三、AI 心理干预类竞品（与 LLM+CBT 定位直接相关）

### 1. Woebot — 先驱者（已关闭，重大警示案例 ⚠️）

- 规则驱动 CBT 对话机器人（非 LLM），14 项 RCT 研究支持
- **2025年6月30日关闭 App**。原因：①FDA 医疗器械审批成本过高；②FDA 尚未制定 LLM 监管路径，想用 LLM 但无法获得许可；③缺乏可持续的 B2C 商业模式
- **对 MindFlow 的核心警示：绝对不要定位为「心理健康治疗」产品。用「行为效率教练」而非「心理健康应用」规避监管风险**
- 来源: statnews.com/2025/07/02; medium.com/@scott-wallace-phd "R.I.P. Woebot"

### 2. Wysa — 当前最成功的 AI 心理支持平台

- CBT/DBT + 规则+LLM 混合模型；NHS 使用；4000 万+ MAU
- **商业路径 B2B 为主**：NHS/保险/企业合作；2025.3 收购 April Health 打通医保报销
- 免费层体验价值 → 临床验证 → 卖给机构客户 = **目前唯一可持续的 AI+CBT 商业模式**
- 来源: wysa.com; bhbusiness.com/2025/03/11; choosingtherapy.com

### 3. Youper — 情绪追踪 AI

- AI 情绪助手 + CBT/ACT/DBT/PST；斯坦福临床验证；300 万+ 用户；~$69.99-89.99/年
- **差异化：实时情绪追踪 → 自适应干预**——根据当前情绪状态推荐不同 CBT 练习
- **MindFlow 可借鉴的设计模式：检测用户状态 → LLM 生成对应干预策略**
- 来源: cnet.com; intuitionlabs.ai; therappai.com

### 4.2 差异化定位验证：「理解拖延原因并主动干预」是否是市场空白？

**结论：是。中等置信度（基于 10+ 款产品功能与用户评价的推断）。**

1. **直接证据**：零竞品提供「为什么你会拖延」的根因分析
2. **最接近者的差距**：Rize AI Coach（描述性，不碰心理学）；Youper（懂情绪，不关心工作）；Sukha（一体化，无因果分析）
3. **用户侧证据**：反复出现的「用一周就没效果」「太死板」抱怨 = 现有产品只提供外部约束，约束被习惯化后效果消失
4. **需验证的风险**：①「理解原因」是否真是付费驱动力；②LLM 做 CBT 类分析的监管风险（Woebot 已证实）；③根因分析结果难以量化为留存指标

**建议：将「根因分析」定位为差异化功能而非核心卖点；将「聪明的自适应干预」作为核心卖点——用户感知的是更好的拦截效果，而不是背后的 CBT 理论。**

### 4.3 可商业化的功能优先级（付费墙设计）

| 优先级 | 功能 | 理由 | 对标参考价 |
|-------|------|------|-----------|
| **P0 核心付费** | AI 个性化专注计划（行为+情绪动态调整） | 无竞品能做好；LLM 天然不适合免费 | Rize Pro ~$28.99/mo |
| **P0 核心付费** | 自适应拦截强度（按当日状态动态调整） | 解决「拦截太死→绕开」矛盾；立项书问题3的答案 | 无直接对标 |
| **P1 高价值** | 深度周报+行为模式洞察 | 付费转化锚点 | RescueTime Solo+ $16/mo |
| **P1 高价值** | 不限次 AI 对话（CBT 式根因对话） | 高价值但 token 成本需控制 | Wysa Premium |
| **P2 转化层** | 多设备同步+历史数据无限保留 | 标准做法 | 几乎所有竞品 |
| **P3 社区层** | 团队/自习室模式 | 粘性功能，早期可不做 | 番茄ToDo |
| **免费开放** | 基础番茄钟、基础拦截、每日简报、限次 AI 对话（10次/日） | 感受价值→限制使用→引导付费 | — |

**早期不建议做**：团队版、API/Webhook、白标、第三方 BI 导出。

### 4.4 对后端能力的具体要求推导

| 后端能力 | 优先级 | 理由 |
|---------|--------|------|
| 用户画像/行为数据库（Event Sourcing 风格） | 必须 | AI 个性化依赖历史行为数据 |
| LLM 编排层（CBT 提示管线+安全护栏） | 核心差异化 | CBT 对话管理、安全输出护栏、对话记忆、上下文管理 |
| 实时/准实时处理 | 高 | nudge 时机需要准实时触发 |
| 通用 REST API | 高 | 多客户端统一后端 |
| 数据导出（CSV/PDF） | 中高 | 缺失会丢失专业用户 |
| WebSocket/SSE | 中 | MVP 可降级为轮询 |
| 团队版 API | 低（二阶段） | 验证核心价值后再做 |
| OAuth2 第三方接入 / 国际化 | 低 | 有需求再做 |

**架构注意事项（竞品教训）：**
1. **安全护栏内建于 LLM 层**：prompt 注入 disclaimer + 输出内容过滤
2. **不用「诊断/治疗」表述**（Woebot 教训）：用「帮你理解工作模式」替代
3. **隐私明确告知**：声明不截图、不录屏、不记录按键（自动追踪易被视为「监视」）
4. **Offline-first 非必须**：基础计时/拦截离线可用，CBT 对话要求在线是可接受的

### 4.5 干预机制设计（立项书关键问题3的竞品答案）

| 机制类型 | 代表产品 | 用户评价 |
|---------|---------|---------|
| 时间盒/番茄钟 | Forest / 番茄ToDo | 初期有效；1周后习惯化 |
| 外部强制拦截 | Cold Turkey / Freedom | 最有效但抗拒感强 |
| 社交问责 | 自习室 | 依赖社交圈活跃度 |
| 损失厌恶 | Forest 树枯萎 | 对部分用户完全无效 |
| 数据反馈 | RescueTime / Rize | 仅对理性自省型有效 |
| AI 教练（描述性） | Rize AI Coach | 好评但不够深入 |
| 情绪自适应干预 | Youper | 有效但仅限情绪场景 |
| CBT 对话式干预 | Wysa / Woebot | 临床有效但 B2C 难盈利 |

**MindFlow 的机会：将「CBT 对话式干预」+「情绪自适应干预」适配到工作/学习场景——现有产品的盲区。**

### 4.6 关键洞察总结

1. **市场空白确认**：无产品同时具备「自动行为追踪 + AI 根因分析（CBT）+ 自适应干预」三要素
2. **定价参考**：Rize 验证「AI 个性化支撑高溢价」（$28.99/mo）；国内建议 15-30 元/月；买断制不适合 AI 产品
3. **最大风险**：监管——定位为「效率增强工具」，不碰医疗/诊断/治疗标签
4. **问题3答案**：无产品做到「根据用户当日状态动态调整干预强度」= 核心入场机会
5. **国内竞品威胁有限**：无 AI/CBT 能力且多年无技术迭代；真正的挑战是用户认知教育成本
